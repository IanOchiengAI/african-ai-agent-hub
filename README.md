# African AI Agent Developer - Obsidian Vault

A complete 6-month roadmap to become an AI agent developer with African domain expertise.

## Quick Start

### 1. Download This Vault
1. Download the entire `african-ai-agents` folder
2. Save it somewhere on your computer

### 2. Open in Obsidian
1. Open Obsidian
2. Click "Open folder as vault"
3. Select the `african-ai-agents` folder
4. Your vault is now ready!

### 3. Start Here
Open `00-PROJECT-MASTER.md` - this is your home base.
